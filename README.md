# Overview

Track your ERC20 shorts against ETH on Uniswap V3.

# Installation

Run `python -m pip install -r requirements.txt`

Dependencies
- Python 3.10
- web3.py 5.25.0
- Flask 2.0.2

# Getting started

Update the example `.env` file with your Infura details.

```
export FLASK_APP=app
flask run
```
